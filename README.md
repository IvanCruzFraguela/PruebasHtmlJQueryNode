# PruebasHtmlJQueryNode

Hay que instalarse node.
Para hacer las pruebas de jQuery es más sencillo que usar las rutas de ficheros.
Toda la parte estática (Html) requiere pocas líneas de código y son cortar y pegar.

npm install express //para las rutas y el objeto app
npm install --save body-parser //para el post con json

ejecutarlo node server.js

Hay ejemplos de peticiones ajax desde jQuery para Post y Get
Estilos en Html que se ajustan al tamaño de la ventana del navegador y se reparten con flex.

No dio tiempo a enviar y recibir json por post. Pero es la idea para todo.

En el node tenemos la página por defecto, petición get y petición post.
